Student API - Request Information

Create Student

Use this to add a new student record to the system.

Method: POST

To access, go to: /api/create-student

Get All Students

Use this to retrieve a list of all students in the system.

Method: GET

To access, go to: /api/get-students

Get Student

Use this to fetch the details of a specific student using their ID.

Method: GET

To access, go to: /api/get-student/{id}

Update Student

Use this to update a student's information in the system.

Method: PUT

To access, go to: /api/update-student/{id}

Delete Student

Use this to delete a specific student's record from the system.

Method: DELETE

To access, go to: /api/delete-student/{id}

Generate Student Summary

Use this to generate a summary for a specific student.

Method: GET

To access, go to: /api/students/{id}/summary
